# Styling Text - Bold, _Italic_ , ~~Strikethrough~~
## 1. Bold
`**text**` or `__text__`  or `ctrl+b`  
**text**

## 2. Italic
`*text*` or `_text_` or `ctrl+i`  
*text* 

## 3. Both bold and italic
`***text here***`  
***Text here***

## 4. Strikethrough

`~~text~~`  
~~text~~
